# pdcurses_examples

To build:

windows:
    cmake -G "Visual Studio 15 2017 Win64" .
linux:
    cmake .

cmake --build .

Examples:

 - 0 - hello world, initialize curses
 - 1 - colors
 - 2 - keyboard usage
 - 3 - output functions
 - 4 - window resize
 - 5 - creating windows 
 - 6 - roguelike character
 - 7 - scan functions
 - 8 - mvchgat
 - 9 - some kind of menu
 - 10 - screen dump/restore
 - 11 - ASC variables , extended encoding
 - 12 - ascii art
 - 13 - drawing lines, rectangles
